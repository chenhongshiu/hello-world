## While 迴圈
問題：while迴圈的執行範圍為何？何時跳回重覆? 注意避免"無窮迴圈"的產生

### 會產生無窮迴圈
```
i=1
while i < 3:
 print(i)
i+=1 #這裡在做 i=i+_1

print("abc")
```
會出現無限的數字1，因為只執行到 print(i) 那行，然後又回到 while 那行，稱無窮迴圈

## for迴圈
```
for x in range(1,10):
  print(x)
  
會 print 出換行
1
2
3
4
5
6
7
8
9
```
如果加入 print(x,end=(''))
```
for x in range(1,10):
  print(x,end=(''))
會 print 出不換行 
123456789
```
